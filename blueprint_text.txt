Three-Tier Adaptive Learning SystemThree independent feedback layers at different timescales. Tier 1 never modifies model weights. Tier 2 nudges the output head only. Tier 3 fully retrains with strict deployment gating.TIER 1 -- Daily Source Trust Weight Update (10:10 AM every trading day)▸ What changes: Trust score per news source per stock. NOT model weights.▸ Logic: If prediction gap improved vs yesterday -&gt; increase trust of sources that contributed▸ Logic: If prediction gap worsened -&gt; decrease trust of contributing sources▸ Bounds: Trust weight hard-clamped to [0.1, 2.0]. No source can dominate or be silenced.▸ Diversity guard: No single source may exceed 40% of total weight for any stock▸ Anomaly guard: Skip update if |actual_move| &gt; 3 sigma historical std (budget day, crash, etc.)▸ Stale decay: Sources inactive 7+ days decay toward 1.0 neutral at 2% per day▸ Storage: trust_weights/{date}.json -- immutable, versioned, committed to Git nightlyTIER 2 -- Weekly EWC Weight Nudge (Every Friday 11:00 PM)▸ What changes: Last 2 FC layers of T-LSTM head ONLY. LSTM backbone and Transformer encoders frozen.▸ Method: Elastic Weight Consolidation (EWC) -- Fisher matrix penalises overwriting critical weights▸ Settings: Learning rate 1e-5 (10x smaller than base), 3 epochs, EWC lambda = 400▸ Fisher matrix: Computed once after each monthly retrain. Reused for all nudges until next retrain.▸ Rollback: If 7-day MAE after nudge &gt; before nudge (+2% tolerance) -&gt; restore previous checkpoint▸ Skip condition: If any day that week triggered the anomaly guard -&gt; skip entire nudge that week▸ Purpose: Adapts to regime shifts without catastrophic forgetting of 20-year historical patternsTIER 3 -- Monthly Full Walk-Forward Retrain (Last Sunday of each month)▸ What changes: Full model retrained from scratch on expanding dataset (all history to current month)▸ Holdout rule: Year 20 NEVER included in training or validation. Sealed until final evaluation only.▸ Validation: New model evaluated on 3-month rolling window before deployment decision▸ Deployment gate: New model MAE must beat live model MAE by &gt;= 2% to replace the live model▸ If gate fails: Live model remains active. New model archived for analysis.▸ Post-deploy: Recompute Fisher Information Matrix on new training set for next months EWC nudges▸ Duration: ~4-6 hours on Machine 1 RTX 5060. Runs Sunday to avoid weekday disruption.EMERGENCY FREEZE -- Set FREEZE_ALL_UPDATES = True in config.yml▸ Disables all three tiers simultaneously. Model still makes predictions but learns nothing.▸ Use on: Union Budget day, election results, surprise RBI decisions, global market shocks▸ Reset manually to False once market normalises -- typically 2-3 trading days later▸ Rationale: Atypical event days break the news-to-price relationship. Learning from them corrupts state.6.
